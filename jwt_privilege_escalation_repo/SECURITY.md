# SECURITY / ETHICS

This repository is an educational localhost lab.

## Allowed Scope
- Testing ONLY on your own machine, on `127.0.0.1`.
- Testing ONLY the code in this repository.

## Not Allowed
- Using these methods on systems you do not own or have explicit written permission to test.
- Publishing real secrets, tokens, or credentials from real systems.

## Reporting
If you adapt this lab for a real organization, follow responsible disclosure policies.
